# App-estacion
Esta aplicación proporciona datos climáticos esenciales en tiempo real.

App-Estación recopila y presenta las siguientes métricas: temperatura, humedad, presión atmosférica, velocidad del viento y riesgo de incendio. Es una herramienta fundamental para el análisis ambiental, ideal para la planificación en agricultura o para cualquier proyecto que requiera información meteorológica precisa y localizada.
